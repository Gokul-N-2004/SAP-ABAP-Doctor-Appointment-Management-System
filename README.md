# Doctor Appointment Management System

## Description
This project is developed to manage doctor appointments between patients and doctors in a clinic environment.  
It allows users to create and manage appointments, track appointment status, and maintain doctor and patient records efficiently.

The system automatically generates a unique **Appointment ID** and allows users to update the appointment status as **Booked**, **Visited**, or **Cancelled**.
